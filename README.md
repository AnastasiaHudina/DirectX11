Здесь представлены практические задания к курсу "Компьютерная графика" с подробными комментариями к коду и скриншотами вывода программы. Название каждой папки соответствует порядковому номеру практического задания.

<br>СПИСОК ЗАДАНИЙ:
1.	Создать оконное графическое приложение
   <br>•	Проинициализировать DirectX
   <br>•	Заливать всё фиксированным цветом на каждом кадре
   <br>•	Поддержать изменение размера окна (поменять размер текстуры в swap chain, если пришло сообщение WM_SIZE)
3.	Добавить рисование треугольника в NDC в графическое приложение
4.	Вывести кубик
  <br>•	Вывести вращающийся кубик в трехмерном пространстве
  <br>•	Добавить управление камерой (можно только вращение)
  <br>•	Передавать m и vp матрицы в отдельных буферах
5.	Текстурирование
  <br>•	Добавить DDS текстуру на кубик
  <br>•	Добавить skybox с cubemap текстурой в сцену
6.	Глубина и порядок
  <br>•	Добавить второй кубик или сферу к сцене
  <br>•	Поддержать буфер глубины формата D32_FLOAT и reversed depth
  <br>•	Добавить прозрачные объекты с правильной сортировкой и без записи в буфер глубины
  <br>•	Сохранить работоспособность skybox
7.	Добавить в приложение точечные источники света и материал с картой нормалей
8.	Добавить в проект instancing для вывода непрозрачных кубиков с разными текстурами и использованием frustum culling
9.	Добавить фильтр постпроцессинга: перевод в оттенки серого, изменение яркости или контраста, сепию или что-то еще
10.	Поддержать frustum culling на GPU
11.	Добавить информацию о количестве отрисованных экземпляров при использовании frustum culling на GPU
